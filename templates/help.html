Help

The ECiLPSE webserver is crafted to assist researchers in predicting enzyme classification for provided protein sequences. This page offers guidance on various modules within the server. The key modules encompass:
Prediction using text box: This feature enables users to paste a set of protein sequences in the FASTA format for the prediction of their respective enzyme classes.
Prediction by uploading file: This feature enables users to upload a file containing sequences. It is essential to ensure that the sequences are formatted in FASTA and saved with the .fasta file extension.
